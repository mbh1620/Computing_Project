Vector tetrahedron::cog_calc ( )
virtual

Center of Gravity redefined because the centroid of a tetrahedron is different.

The Tetrahedron centre of gravity calculation is calculated by adding all the points up and then dividing by 4. This is assuming that the density of the tetrahedron is constant through out.

Reimplemented from cell.

float tetrahedron::volume_calc ( )
virtual

Volume function redefined because the volume of a tetrahedron is different.

The volume of a tetrahedron for four points can be defined as:

V = [(a-d)dot((b-d)cross(c-d))]/6 V = [xdot(ycrossz)]/6

dot and cross functions have been defined in the vector.hpp library so we can use them.

Reimplemented from cell.
